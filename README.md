# undercoin
Bitcoin utility belt

### usage

```bash
npm install undercoin
```
In node or browser (ie- via Browserify)
```javascript
var undercoin = require('undercoin')
```


### api

** btcToSatoshi **  
`undercoin.btcToSatoshi(btcAmount)`  
Converts BTC to satoshis

```javascript
undercoin.btcToSatoshi(1) // 100000000
```
